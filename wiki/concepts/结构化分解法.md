# 结构化分解法

高善文在处理复杂系统问题时的核心技术手段：将庞大的总量问题拆解为不同层级的核心驱动要素，找到当前阶段的"关键变量"。

---

## 核心思想

> 宏观研究如果只停留在GDP、CPI等总量指标上，往往会失去解释力。有效分析必须触及**中观行业层面**和**微观企业决策层面**。

---

## 三层结构

```
宏观层：GDP / CPI / 投资增速 / 货币供给
        ↓ 解释力不足时向下拆解
中观层：基建投资 / 制造业投资 / 房地产投资
        ↓ 进一步追问驱动力
微观层：财政政策 / 城投融资环境 / 产能利用率 / 土地政策 / 人口流入预期
```

---

## 典型案例：投资增速分解

| 子项 | 核心驱动力 |
|------|-----------|
| 基建投资 | 财政政策 + 城投平台融资环境 |
| 制造业投资 | 全球需求周期 + 企业产能利用率 |
| 房地产投资 | 土地政策 + 信贷环境 + 人口流入预期 |

三个子项背后是**完全不同的逻辑**，混在一起看"投资增速"会掩盖真相。

---

## 核心变量识别原则

不同经济阶段，主导经济运行的核心变量不同：
- 有时是**汇率**
- 有时是**产能周期**
- 有时是**信贷供给**

研究者的任务：通过逻辑推演和证据排查，**准确找到当前阶段的那个关键变量**，对其进行穿透式研究。

---

## 操作步骤

```
1. 识别总量指标的异常或失去解释力的信号
2. 将总量拆解为2-4个有不同逻辑驱动的子项
3. 对每个子项独立分析其核心驱动力
4. 判断当前阶段哪个子项是主导变量
5. 对主导变量进行多维交叉验证
```
