## 核心论点（5条）

1. **摒弃类比与阴谋论**：类比法直观但扭曲，阴谋论无法证伪，二者都是科学研究的大敌，必须用严密逻辑和经济学因果律取代
2. **逻辑框架先行**：逻辑是观察市场的雷达，资产重估理论、产能周期、潜在增速框架都是为了理清特定时期的经济主线
3. **实证主义锚点**：结论必须可被证伪；用工业用电量、港口吞吐量等物理指标交叉验证官方数据
4. **结构化分解**：总量指标（GDP/CPI）失去解释力时，必须下沉到行业层面和企业决策层面找核心驱动变量
5. **分析师是决策辅助者**：价值在于提供完整逻辑框架供决策者使用，而非提供点位预测

## 关键事实与数据

- 高善文学术背景：清华大学 + 北京大学，导师为央行前行长周小川
- 资产重估理论奠基于2005-2007年大牛市，核心逻辑：经常账户顺差→基础货币被动投放→企业存款从定期转活期→流动性涌入资产市场
- 2010年前后逆主流预判中国经济增速长期下降，后被历史验证
- 代表作：《经济运行的逻辑》

## 方法论技术手段

| 技术 | 说明 |
|------|------|
| 追根溯源 | 在历史案例中定位当前现象，寻找共性驱动机制 |
| 经典迁移 | 将成熟经济学模型迁移到中国，根据制度约束修正 |
| 多维交叉验证 | 分析通胀时同时看CPI/PPI、企业毛利润、开工率、工资增速 |
| 结构化分解 | 投资增速→基建/制造业/房地产，各有不同逻辑驱动 |
